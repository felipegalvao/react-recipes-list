# React Recipes List

React app to show recipes in a organized way.

Demo: http://felipegalvao-recipes-list.surge.sh/

## What I Used

- React
- Express
- webpack
- Babel
- Axios
- Jest
- Enzyme

## Local Installation Instructions

To run this project locally, follow these steps:

- Clone this repository
- Run "npm install"
- Run "npm start"
